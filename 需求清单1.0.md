# Mellow 前端重建需求清单

> **原则零：绝不自创 UI。所有组件、动画、特效必须来自 Flutter Vignettes 源码或 pub.dev 高星包。**
>
> **视觉北极星：Flutter Vignettes (https://github.com/gskinnerTeam/flutter_vignettes)**
>
> 后端 API 全部跑通，前端可全量对接。

---

## 〇、基础决策

| 决策项 | 选择 | 理由 |
|--------|------|------|
| 状态管理 | Riverpod（简化，不用 codegen） | 比 Provider 更现代，但避免 @riverpod 生成代码的复杂度 |
| 路由 | GoRouter 声明式路由 | 支持嵌套导航、重定向、deep link |
| 模型 | 纯 Dart class + fromJson/toJson | 不用 Freezed，减少生成文件 |
| 设计系统 | shadcn_ui（基础组件）+ Vignettes（特效组件） | shadcn 负责按钮/输入框/卡片，Vignettes 负责动画/粒子/3D |
| 导航 | bubble_tab_bar（Vignettes 原版） | 图标 3D 翻转 + 弹性缩放 + 双层阴影 |
| 主题 | Light + Dark 双主题 | 用 dark_ink_transition（Vignettes 原版）过渡 |
| 品牌色 | 绿 #5BC91A + 橙 #F09433 | 成长感 + 温暖鼓励 |
| 字体 | Lato（正文）+ Quicksand（标签/按钮） | Google Fonts |

## 三、认证流程

- [ ] **3.1 启动页 SplashScreen**
  - CustomPaint 小猫 Logo + "你的 AI 英语老师"
  - 自动调用 `/api/v1/auth/me` 检测 JWT 是否有效
  - 有效 → 跳转 `/chat`
  - 无效 → 显示"开始"按钮 → 跳转 `/auth/login`
  - 右下角主题切换按钮（dark_ink_transition）
- [ ] **3.2 登录页 LoginScreen**
  - plant_forms 风格输入框（邮箱 + 密码）
  - 品牌绿登录按钮（带 loading 态）
  - 客户端验证（邮箱格式、密码非空）
  - 调用 `POST /api/v1/auth/login` → 存储 token → 跳转聊天
  - 底部"没有账号？立即注册 →"链接
  - 错误提示（红色文字）
- [ ] **3.3 注册页 RegisterScreen**
  - plant_forms 风格输入框（昵称 + 邮箱 + 密码 + 确认密码）
  - 客户端验证（邮箱正则、密码≥6位、两次一致）
  - 调用 `POST /api/v1/auth/register` → 自动登录 → 跳转角色选择
  - 底部"已有账号？去登录 →"链接
- [ ] **3.4 自动登录**
  - 启动时读取 SharedPreferences 中的 JWT token
  - 调用 `/api/v1/auth/me` 验证

  - 失效 → 清除 token，显示登录页
- [ ] **3.5 退出登录**

  - 跳转登录页
